# Homebrew Tap for meow-meow-meow

This is a Homebrew tap for the enhanced meow-meow-meow fork that allows you to type "meow meow meow" to print ASCII cats!

## Installation

Users can install meow-meow-meow with a single command:

```bash
brew install starwaves1/meow-meow-meow/meow-meow-meow
```

Or they can tap the repository first and then install:

```bash
brew tap starwaves1/meow-meow-meow
brew install meow-meow-meow
```

## Usage

After installation, you can use the enhanced meow features:

```bash
# Print one cat
meow

# Print three cats by typing "meow" three times
meow meow meow meow

# Use the count flag
meow -c 5

# Be literally this cat
meow -l
```

## Uninstallation

```bash
brew uninstall meow-meow-meow
brew untap starwaves1/meow-meow-meow
```
